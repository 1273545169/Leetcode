

### 题目描述

[169. 多数元素](https://leetcode.cn/problems/majority-element/)

### 思路解析

投票法，相同+1， 不同-1，若cnt为0，candidate取当前值

```python
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        candidate, cnt = nums[0], 1
        for i in range(1, len(nums)):
            if cnt == 0:
                candidate = nums[i]

            if candidate == nums[i]:
                cnt += 1
            else:
                cnt -= 1
        return candidate
```
